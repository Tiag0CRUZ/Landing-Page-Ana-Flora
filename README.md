 Clínica de Nutrição & Estética

Projeto de um site institucional moderno para clínica de nutrição e estética, desenvolvido com foco em design elegante, navegação fluida e experiência do usuário.

---

Demonstração

> Interface responsiva com múltiplas seções:

* Home
* Clínica
* Tratamentos
* Contato

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)

---

Estrutura do Projeto

```
📦 projeto-clinica
 ┣ 📜 index.html
```

 Funcionalidades

* ✅ Navegação dinâmica entre páginas sem reload
* ✅ Layout moderno e elegante (dark + gold theme)
* ✅ Menu com dropdown
* ✅ Seções organizadas (Hero, Especialidades, Tratamentos, Contato)
* ✅ Botões de ação (WhatsApp / Agendamento)
* ✅ Responsividade para dispositivos móveis
